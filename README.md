# Quilting

This repository collects small hobby projects around quilting.

These projects are built for personal use and have not been extensively tested. They are all designed for macOS, generally using Swift, SwiftUI, and Xcode.

## Projects

- `Foundations`: a macOS Swift project for foundation paper piecing template experiments.
- `Quilt Log`: a macOS SwiftUI app for tracking quilts in a local SQLite database.
